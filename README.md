# 🔐 Encrypted File Vault

A web-based cybersecurity project built with Flask and Python that securely encrypts and decrypts files using password-based cryptography.

## Features

✅ File upload support
✅ File encryption
✅ File decryption
✅ Password-based key generation
✅ Download encrypted/decrypted files
✅ Blue-black cyber UI

## Tech Stack

* Python
* Flask
* Cryptography
* HTML
* CSS
